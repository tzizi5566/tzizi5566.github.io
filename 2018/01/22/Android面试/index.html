    running/paused/stopped/killed

  2. activity生命周期
    Activity启动 -> onCreate() -> onStart() -> onResume()
    点击Home键回到主界面(Activity不可见) -> onPause() -> onStop()
    当再次回到原Activity时 -> onRestar() -> onStar() -> onResume()
    退出当前Activity时 -> onPause() -> onStop() -> onDestroy()
