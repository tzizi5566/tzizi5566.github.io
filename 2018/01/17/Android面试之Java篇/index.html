Java面向对象的三个特征与含义

Java面向对象的三个特征为：封装、继承、多态。

封装：把对象的属性和方法封装到一个类中，对外只提供需要的属性和方法，还可以通过Java的四种访问修饰符对访问的范围进行限制。

继承：子类对象可以继承父类的属性和方法，父类拥有的属性和方法子类同样拥有，子类还可以在此基础上扩展自身特有的属性和方法。

多态：多态的存在有三个前提 1要有继承关系。2子类要重写父类的方法。3父类的引用指向子类对象。
一个方法在不同的对象中调用会有不同的结果。
